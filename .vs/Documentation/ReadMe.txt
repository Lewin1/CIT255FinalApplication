Title: Movie Database

Description: An app that edits a database of movies.

Application Type: WinForms

Persistence Type: XML

Persistence Location: Data File(XML) in the bin folder

Author: Nicholas Lewin

Dated Created: 11/25/2017

Last Modified: 12/32017

Instructions for use: 

When the app starts, the movies that are already in the XML file will load.
Use the Add button to add movies.
Select a movie in the listbox and then click on the Update button to update movies.
Use the Delete button to delete movies by their ID.
Use the vote buttons to vote up or down movies.
Use the query buttons to sort through the data by Genre, Name, or Rating.
